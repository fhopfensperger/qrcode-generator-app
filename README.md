A simple web app to generate QR-Codes for vCards or URLs

## Run it

1. Start the container
  ```bash
  docker run -p 8080:8080 --rm quay.io/fhopfensperger/qrcode-generator-app
  ```

2. Open Web browser [localhost:8080](http://localhost:8080)
